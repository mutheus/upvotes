### 🏃‍♂️ Run on your machine

Clone the project and access the folder:

```bash
git clone https://github.com/mutheus/upvotes.git && cd upvotes
```

Install the dependencies:
```bash
yarn install
```

Then, run the development server:

```bash
yarn start
